AI 기반 트리플 협업 코인 프로젝트.  
GPT + Claude + Groq 구조로 구성된 초고속 자동화 시스템.

## 🧩 구성

- `core/`: 공통 유틸 및 베이스 컴포넌트
- `data/`: API 연동 및 모델 정의
- `domain/`: 비즈니스 로직 및 리포지토리
- `presentation/`: UI, 상태관리, 라우팅 구성

## ✅ 기능 목표

- 실시간 코인 데이터 분석 및 시각화
- 3단계 AI 협업 구조로 구성된 전략 로직
- 자동화 트레이딩 & 실시간 이벤트 처리
